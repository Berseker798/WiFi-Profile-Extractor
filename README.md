WiFi Profile Extractor (Educational)
🇵🇱 Polski (PL)
To narzędzie zostało stworzone w celach edukacyjnych, aby zrozumieć, jak system Windows przechowuje dane o sieciach Wi-Fi i jak można je automatycznie wyodrębnić za pomocą prostych skryptów.

Działanie: Skrypt wykorzystuje wbudowane narzędzie Windows netsh do wyszukania zapisanych profili Wi-Fi, a następnie wyciąga z nich szczegółowe informacje (w tym klucze zabezpieczeń w formacie czystego tekstu).

Instrukcja: Uruchom plik hasla.bat jako Administrator. Wyniki zostaną zapisane w folderze hasla_wifi.

Disclaimer: Narzędzie wyłącznie do użytku w celach edukacyjnych na własnym sprzęcie.

🇬🇧 English (EN)
This tool was created for educational purposes to understand how Windows stores Wi-Fi network profiles and how they can be automatically extracted using simple scripts.

How it works: The script uses the built-in Windows netsh utility to search for saved Wi-Fi profiles and then extracts detailed information from them, including security keys in clear text.

Instructions: Run the hasla.bat file as an Administrator. The results will be saved in the hasla_wifi folder.

Disclaimer: For educational purposes on your own hardware only.

🇩🇪 Deutsch (DE)
Dieses Tool wurde zu Bildungszwecken entwickelt, um zu verstehen, wie Windows WLAN-Profile speichert und wie diese automatisch extrahiert werden können.

Funktionsweise: Das Skript verwendet das Windows-Tool netsh, um gespeicherte WLAN-Profile zu suchen und Details (einschließlich Passwörter im Klartext) zu extrahieren.

Anleitung: Führe hasla.bat als Administrator aus. Die Ergebnisse werden im Ordner hasla_wifi gespeichert.

Haftungsausschluss: Nur zu Bildungszwecken auf eigener Hardware verwenden.
